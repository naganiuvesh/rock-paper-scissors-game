# Rock Paper Scissors Game 🎮

A simple **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.

## Features

* 🪨 Rock, 📄 Paper, and ✂️ Scissors buttons
* 🤖 Random computer choice
* 📊 Live score tracking

  * Your Points
  * Computer Points
* 🏆 Displays the game result:

  * Win
  * Lose
  * Draw

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Play

1. Click on **Rock**, **Paper**, or **Scissors**.
2. The computer will randomly choose its move.
3. The winner is decided based on the game rules.
4. Scores are updated after every round.

## Project Purpose

This project was created to practice JavaScript fundamentals such as DOM manipulation, event handling, conditional statements, and random number generation.
